# Youtube AI Clipper
There will be no UI.
Main target is to produce N number of clips from Youtube URL.
Claude code does the job by working from this repo using command and available code / tools in this repo.

E.g.:
"Clip this https://www.youtube.com/watch?v=ncXkv77uVH4 to produce 10 meaningful messages or insight clips."
You will download the video, transcribe it, then clip it as prompts, make sure to remove the idle moments between sentences.
